# Logo Carousel Widget for Elementor

A drag-and-drop supported, smooth scrolling, pure CSS marquee logo showcase plugin for WordPress, specifically built for Elementor.

![Logo Carousel Demo](assets/REC-20260405144014.gif)

## Features
- **Infinite Marquee:** Continuous CSS-based scroll effect.
- **YouTube Support:** Logos can trigger a lightning-fast lightbox popup.
- **Bi-directional:** Define the direction (Left-to-Right or Right-to-Left), and let multiple rows automatically scroll in opposition.
- **Fully Customizable:** Tweak opacity, hover effects, gaps, sizes, colors and CSS edge fades directly from Elementor.
- **Lightweight:** Loads JS/CSS asynchronously and only when the widget is present on the page.

## Requirements
- WordPress 5.0+
- Elementor 3.0+
- PHP 7.4+

## Installation
1. Download the latest release `.zip` from GitHub.
2. Inside your WordPress admin dashboard, navigate to **Plugins > Add New > Upload Plugin**.
3. Select the downloaded `.zip` file and install.
4. Activate the plugin.
5. In your Elementor editor, search for "Logo Carousel" and drag it onto your page.
